\section*{Assignment}

\noindent
The goal of this assignment is to create a program that accepts two numbers from the command line and prints their sum. This exercise wil familiarize you with the C++ integral types, command line arguments, and the \texttt{iostream} library.

\subsection*{Requirements}

\begin{enumerate}
	\item The program should accept exactly two command line arguments.
	\item The program should:
	\begin{itemize}
		\item Convert the command-line arguments (which are strings) to integers.
		\item Calculate the sum of the two integers.
		\item Output the sum to the console.
	\end{itemize}
\end{enumerate}

\noindent
You may assume all input is valid.

\section*{Submission}

\noindent
Submit the C++ source file (named \texttt{add.cpp}) containing your solution.

\section*{Sample Output}

\begin{verbatim}
>> ./add 3 5
8

>> ./add -1 1
0

>> ./add -4 -4
-8
\end{verbatim}
