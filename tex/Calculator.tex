\section*{Assignment}

\noindent
In this assignment, you will write a simple program that takes two numbers as input and prints their sum, their difference, and their product. This program will help to familiarize you with C++ developer tools, types and variables, and the \texttt{iostream} library.

\section*{Requirements}

\begin{enumerate}
	\item The program should:
	\begin{enumerate}
		\item Accept exactly two integers, $a$ and $b$, from standard input.
		\item Compute the sum ($a+b$), the difference ($a-b$), and the product ($a\times b$) of the two numbers.
		\item Output results according to the format shown in the sample output.
	\end{enumerate}
\end{enumerate}

\section*{Submission}

\noindent
Please submit a single C++ file (named \texttt{calculator.cpp}) with your solution.
